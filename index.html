dGrowl::Projects
Thanks for visiting! This is a home for some of the major personal projects that I've developed. Each was motivated by a mixture of three reasons: to feed my own interest, to solve problems that I encountered, and to learn how to use new tools. I dig into these kinds of programs whenever I have extra time to spend (usually between school quarters).
HopSlash
 — C++, SDL, rapidjson

HopSlash is a demo game built on top of a custom engine that supports rendering, input, animations, and physics. The player controls a hero that can run, jump, and attack inside a side-scrolling world of platforms. I had learned a lot about SDL and wanted to explore using its capabilities with a real project. A major goal of the engine was for it to be as versatile as possible, so I stored settings, terrain, textures, and animation data in external files (using JSON because it was easy to prototype with and to edit by hand).

Art was done by the amazing rvros.

Repository: github.com/dGrowl/HopSlash
Download (x64): HopSlash_x64.zip

ShortSeek
 — Python, Tkinter

ShortSeek is a Python script that generates optimal search terms for a given list of strings. It can be used from the command line (sseek.py) or from a GUI (ss_gui.py), and has a few different options used to configure its output. I wrote it because I was searching through a lot of lists and became curious to know what the shortest possible input strings would be to yield the results I was interested in. It can also load from (and write to) files.

Repository: github.com/dGrowl/ShortSeek
Download: ShortSeek.zip

SiteDeltaMonitor
 — C++, Qt, gumbo

SiteDeltaMonitor runs quietly as a system tray icon, waiting to announce any detected changes to websites. It has separate windows that open for configuration and reporting. It targets URLs and can select specific HTML identifiers as well, thanks to the gumbo HTML parsing library. I wrote SDM because I wanted an alert whenever my favorite digital artists announced that they were accepting commissions.

Repository: github.com/dGrowl/SiteDeltaMonitor
Download (x64): SDM_x64.zip
