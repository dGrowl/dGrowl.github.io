dGrowl::Projects
Thanks for visiting! This is a home for some of the major personal projects that I've developed. Each was motivated by a mixture of three reasons: to feed my own interest, to solve problems that I encountered, and to learn how to use new tools. I dig into these kinds of programs whenever I have extra time to spend (usually between school quarters).
HopSlash
 — C++, SDL, rapidjson

HopSlash is a small, simple demo game meant to demonstrate the current capabilities of an engine that I developed alongside it. It currently supports:

  • Input
  • Versatile entities
  • Animation (using spritesheets)
  • Collision
  • Terrain (using tilemaps)

Art done by rvros.

Repository: github.com/dGrowl/HopSlash
Download (Windows x64): HopSlash_x64.zip

ShortSeek
 — Python, Tkinter

ShortSeek is a script that generates optimal search terms for a given list of strings.

  • Use from the command line (sseek.py) or a GUI (ss_gui.py)
  • Includes options to allow users to fine-tune their results
  • Supports file input and output

Repository: github.com/dGrowl/ShortSeek
Download: ShortSeek.zip

SiteDeltaMonitor
 — C++, Qt, gumbo

SiteDeltaMonitor is a cross-platform desktop app that monitors websites for changes.

  • Runs in the system tray
  • Uses separate windows for configuration and reporting
  • Allows targeting specific HTML elements or entire pages
  • Compares live sites with previous data

Repository: github.com/dGrowl/SiteDeltaMonitor
Download (Windows x64): SDM_x64.zip
